# DisposalGrid Monorepo

A comprehensive waste management and disposal facility information system.

## Repository Structure

- **`/site`** - Next.js frontend website
- **`/etl`** - Extract, Transform, Load pipelines for data processing
- **`/scrapers`** - Web scrapers for collecting disposal facility data
- **`/scripts`** - Utility scripts and automation tools

## Getting Started

Each directory contains its own README with specific setup instructions.
